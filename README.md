# TaskMaster 📝

TaskMaster es una aplicación web intuitiva diseñada para la gestión eficiente de tareas diarias. Este proyecto combina una interfaz limpia con funcionalidades avanzadas para ayudar a los usuarios a organizar su flujo de trabajo de manera efectiva.

🚀 **Ver Demo en Vivo:** https://gomezsl.github.io/mi-gestor-tareas/

---

## ✨ Características Principales

* **Gestión Completa de Tareas:** Crea, edita y elimina tareas fácilmente.
* **Priorización:** Clasifica tus pendientes por niveles de importancia (Baja, Media, Alta).
* **Fechas de Vencimiento:** Mantén el control de tus plazos con una columna dedicada a las fechas límite.
* **Integración con Base de Datos:** Persistencia de datos mediante **Supabase**, asegurando que tu información esté siempre disponible.
* **Interfaz Responsiva:** Diseño adaptado para una experiencia fluida tanto en escritorio como en dispositivos móviles.
* **Contacto Directo:** Botones de acción rápida para comunicación y soporte.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3 y JavaScript (ES6+).
* **Backend / Base de Datos:** [Supabase](https://supabase.com/) (BaaS).
* **Hosting:** GitHub Pages.

## 📦 Instalación y Uso Local

Si deseas ejecutar este proyecto localmente para desarrollo:

1.  Clona el repositorio:
    `git clone https://github.com/Gomezsl/mi-gestor-tareas.git`
2.  Accede a la carpeta del proyecto:
    `cd mi-gestor-tareas`
3.  Abre el archivo `index.html` en tu navegador preferido.

> **Nota:** Para que la base de datos funcione localmente, asegúrate de configurar correctamente tus claves de API de Supabase en el archivo correspondiente.
